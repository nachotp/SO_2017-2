Antes de ejecutar:
	La tarea está hecha en C++ y utiliza el copilador g++

Para compilar correr comandos:
	make: compilará y ejecutará la tarea
	make clean: eliminará los archivos objeto y ejecutable

Tras esto se pueden ejecutar los siguientes comandos:

	mostrar: muestra el contenido del directorio actual
	mover: te permite mover por directorios, usar ".." para retroceder.
		ej: mover Rock
	abrir: muestra los detalles de una canción, usar el nombre del archivo, se debe estar en el directorio correcto.
		ej: abrir: Sulk.mp3
	azar: entrega las canciones de un artista al azar.
	salir: cierra el programa.
